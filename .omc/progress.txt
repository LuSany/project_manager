## API 端点

- `DELETE /api/v1/admin/projects/[id]`
- 权限：仅 ADMIN
- 响应：`{ success: true, data: { id }, message }`

## Learnings

- API 客户端已有 `api.delete` 方法，直接使用
- Prisma 的 `project.delete` 会自动级联删除关联数据
- 前端使用 `confirm()` 简单确认对话框
