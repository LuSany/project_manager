## 💡 关键发现

### SQLite 架构限制

**问题**:
```typescript
// 测试中使用 beforeEach 创建数据
beforeEach(async () => {
  const user = await prisma.user.create({...})
})

// SQLite 不支持：
await prisma.$executeRaw`BEGIN`
await prisma.$executeRaw`ROLLBACK`

// 导致：
// 1. 数据库锁定
// 2. 数据污染
// 3. 唯一性约束冲突
```

**解决方案**:
1. **迁移 PostgreSQL** (推荐) - 4-6 小时
2. **实现 Mock 策略** - 8-10 小时
3. **保持现状** - 当前采用
