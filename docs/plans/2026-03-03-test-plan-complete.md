## 五、运行测试

### 5.1 Mock 工具测试（无需数据库）

```bash

```

### 5.2 完整测试（需要 PostgreSQL）

```bash
# 启动数据库后运行
npm run test:unit
npm run test:integration
npm run test:coverage
```

